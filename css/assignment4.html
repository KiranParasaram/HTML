What is css?

  1. css stand for cascading styles sheet
  2. css describes how HTML elements are to be displayed on screen, paper,or in other media.
  3. css is not neccesary css saves a lot of work.it can control the layoit of multiple web pages all at once
  4. External style sheets are stored in css files.

css can be adde to HTML documents in 3 ways:

  • inline-by using the style attribute inside HTML elements
  • Internal- by using a style element in a head section
  • External-by using a link element to link to an external css file.
  • The most common way to add Css,is to keep the styles in external CSS file.However,in this tutorial inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself.

When comparing css CLASS vs ID , the difference is that CSS class applies a style to multiple elements.ID,on the aother hand,applies a style to one unique element.Id is also special in that you can use a special url to link directly to an element and its used by JavaScript

In CSS,selectors are used to target a special elemts or range of elements on web page.Once an elemnt has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors available.Two of the most commenly used are class and ID. Both are used to target elements to which a style should be applied.
